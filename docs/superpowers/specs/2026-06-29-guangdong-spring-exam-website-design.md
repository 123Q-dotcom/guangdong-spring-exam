# 广东省春季高考（依学考）信息网站 — 设计文档

> 创建日期：2026-06-29
> 状态：已定稿

---

## 一、项目概述

面向考生和家长的广东省春季高考（依学考）一站式信息网站，覆盖政策解读、报名流程、考试科目、分数线、院校招生、备考资料六大方向，支持搜索和筛选等轻度交互。

## 二、用户定位

- **主要受众**：广东省高三学生及其家长
- **使用场景**：快速了解春季高考政策、查询历年分数线和院校招生信息、获取备考指引
- **设备**：PC 和手机端均可访问

## 三、视觉设计 — 苹果极简风格

### 3.1 设计哲学

- 大面积留白，信息有呼吸感
- 每个视觉区块只传达一个核心信息
- 用大号数字和数据本身作为视觉元素
- 微动效和过渡动画让交互自然流畅

### 3.2 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#0071E3` | Apple 蓝，仅用于 CTA、链接、关键强调 |
| 主背景 | `#FFFFFF` | 纯白 |
| 分区背景 | `#F5F5F7` | 浅灰，交替用于区分内容区域 |
| 主文字 | `#1D1D1F` | 深空灰 |
| 辅助文字 | `#86868B` | 中灰 |
| 卡片阴影 | `rgba(0,0,0,0.04)` | 极淡，悬停时加深至 0.08 |

### 3.3 字体

- 系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`
- 标题：加粗（600-700），字距 `letter-spacing: -0.02em`
- 正文：行高 1.6-1.8
- 大数字展示：`font-weight: 700`，超大字号

### 3.4 组件风格

| 组件 | 样式 |
|------|------|
| 导航栏 | 毛玻璃效果（`backdrop-filter: blur(20px)`），半透明背景，固定顶部 |
| 按钮 | 胶囊形 `border-radius: 980px`，主按钮蓝底白字，次按钮白底蓝边框 |
| 卡片 | `border-radius: 20px`，白底极淡阴影，hover 上浮 4px + 阴影加深 |
| 输入框 | `border-radius: 12px`，浅灰边框，聚焦时蓝色边框 |
| 图标 | 线性 SVG 图标 |

### 3.5 动效

- 卡片 hover：`translateY(-4px)` + 阴影加深，`transition: 0.3s ease`
- 内容区进入视口：`fadeInUp` 动画
- 页面切换：`opacity` 淡入 + `translateY(8px)` → 0，`transition: 0.4s ease`
- 搜索框展开：宽度平滑过渡

## 四、信息架构

### 4.1 页面结构

```
首页 (/)
├── Hero 区：大标题 + 副标题 + 搜索栏
├── 春季高考简介卡片
├── 政策速递（最新 3 条）
└── 快捷入口（6 大板块导航卡片）

政策解读 (/policy.html)
├── 页面标题 Hero
├── 政策分类卡片列表
└── 常见问题 FAQ 手风琴

报名流程 (/registration.html)
├── 页面标题 Hero
├── 时间节点时间轴
├── 报名步骤详解
└── 所需材料清单

考试科目 (/subjects.html)
├── 页面标题 Hero
├── 科目筛选标签
└── 各科卡片（大纲、分值、时间）

分数线 (/scores.html)
├── 页面标题 Hero
├── 筛选栏：年份 / 院校 / 类别
└── 分数线表格

院校招生 (/schools.html)
├── 页面标题 Hero
├── 搜索 + 筛选：地区 / 类型
└── 院校卡片列表

备考资料 (/resources.html)
├── 页面标题 Hero
├── 推荐教材
├── 真题下载链接
└── 复习建议
```

### 4.2 全局组件

- **导航栏**：Logo + 7 个页面链接 + 搜索图标（展开搜索框）
- **搜索**：全站关键词搜索，弹出式搜索结果
- **页脚**：版权信息 + 声明
- **返回顶部**：滚动到一定距离后显示

## 五、数据驱动设计

所有可更新数据以 JSON 文件存放于 `data/` 目录：

```
data/
├── policies.json       # 政策内容
├── timeline.json       # 报名时间节点
├── subjects.json       # 考试科目信息
├── scores.json         # 历年分数线
├── schools.json        # 院校招生信息
└── resources.json      # 备考资料
```

数据更新时只需编辑对应 JSON 文件，无需改动 HTML。

## 六、交互功能

| 功能 | 实现方式 | 适用范围 |
|------|----------|----------|
| 全站搜索 | 汇总所有 JSON 数据建索引，前端匹配 | 全局 |
| 科目筛选 | 点击标签过滤，数据驱动 | 考试科目页 |
| 分数线筛选 | 下拉/标签联动筛选表格行 | 分数线页 |
| 院校搜索+筛选 | 输入框搜索 + 地区/类型标签过滤 | 院校招生页 |

## 七、响应式设计

- **断点**：768px（平板/手机分界）
- 移动端导航改为汉堡菜单
- 卡片从多列变为单列
- 表格在移动端改为卡片式展示
- 字体和间距等比缩放

## 八、技术方案

- **技术栈**：纯 HTML + CSS + Vanilla JavaScript
- **依赖**：零外部依赖
- **兼容性**：现代浏览器（Chrome、Firefox、Safari、Edge 近两年版本）
- **部署**：静态文件托管，任意 HTTP 服务器或直接打开
